# 术语表

## PCK

Godot 的资源包格式，发布时常用于打包场景、贴图、材质、脚本和其他资源。

## GDExtension

Godot 4 的原生扩展机制，用于接入 C/C++ 编写的运行时扩展模块。

## AtlasTexture

Godot 中从一张大图集切出某个区域使用的纹理描述。

## Card Pool

卡牌池。项目里不同职业、颜色或用途的卡牌集合。

## sts2

杀戮尖塔2的简称

## spine

这是一个非常流行的 2D 骨骼动画工具，包括明日方舟的战斗小人还有邦多礼手游的糖氏小人，以及杀戮尖塔2的角色和怪物都采用的是这个技术。
本游戏的spine版本是4.2。

### Spine Skeleton

Spine 的骨骼动画数据文件，可为 JSON 或二进制 skeleton 格式。
