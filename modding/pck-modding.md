# PCK Mod

## 适合改什么

PCK 更适合做资源层覆盖，例如：

- 贴图
- 场景
- 主题
- 本地化文本
- 一部分材质与 shader 配置

## 核心原则

最重要的一条是：**替换资源必须与游戏原有资源标识对齐。**

这并不要求站点读者先看到维护者本地目录，但你需要理解它背后的规则：游戏运行时会按既有资源标识加载内容，所以你的替换包必须在命名与组织上和原资源保持一致。

## 实操思路

1. 先确定你要替换的是贴图、场景、文本还是材质
2. 再确认这个资源在原项目中的归属和用途
3. 准备同名、同用途的替换资源
4. 打包为模组 PCK
5. 在游戏中验证是否成功覆盖

## 什么时候优先选 PCK Mod

如果你的目标是：

- 换卡图
- 换 UI 外观
- 换背景或部分场景资源
- 调整本地化文本

那通常应先从 PCK 路线开始，而不是一上来就进 DLL。
